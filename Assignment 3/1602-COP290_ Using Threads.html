Using Threads

Using Threads

 

The purpose of this assignment is to give you some general experience in writing shared memory programs using the standard pthreads library package. For more information on pthreads, you may want to consult the pthreads man page and this tutorial.  Also learn about gnuplot. An example to use gnuplot from within your programs is given here.

 

 

Addition of n numbers

 

We have an array of size n and we want to compute the sum. Sequentially, this takes n-1 steps, but if we can do things in parallel, we can reduce this to log2n steps. Implement this using pthreads. Plot the time taken for different number of threads using gnuplot.  

 

 

Matrix Multiplication

In this assignment you you are to write a version of matrix multiply that uses threads to divide up the work necessary to compute the product of two matrices. A nice solution of this problem is here. Understand the solution and you will implement a variation of this in class.

 

Your program will be invoked using this command:

matrix_multiply -a matrixA -b matrixB -t thread_count

where the -a and -b parameters specify input files containing matricies and thread_count is the number of threads
